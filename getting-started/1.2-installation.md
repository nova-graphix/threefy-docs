# 1.2 Installation

### Requirements

Threefy 2.0 runs on the three.js WebGPU renderer, so it requires recent versions of its peer dependencies:

| Package | Version |
| --- | --- |
| `react` / `react-dom` | `>= 19.1` |
| `three` | `>= 0.185` |

### Install

Install threefy by entering the following command in your project folder.

```sh
npm install threefy
```

Threefy keeps three.js and react as external peer dependencies (they are not bundled), so it is recommended to install them explicitly alongside threefy:

```sh
npm install threefy three react react-dom
```

\*NOTE: If three.js and react are already installed in your project, they will be reused. Threefy imports `three/webgpu` directly, so no bundler aliases or extra build configuration are required on your side.
