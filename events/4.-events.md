# 4. Events

The following code briefly shows how to handle events occurring in a 3D model. As you can see from the example below, it is very clear and simple. The table below describes the events handled by threefy.

<table><thead><tr><th width="339">Event</th><th>Occurs When</th></tr></thead><tbody><tr><td>onClick</td><td>The user clicks on a 3D element</td></tr><tr><td>onContextMenu</td><td>The user right-clicks on a 3D element</td></tr><tr><td>onDoubleClick</td><td>The user double-clicks on a 3D element</td></tr><tr><td>onWheel</td><td>The mouse wheel is rolled over a 3D element</td></tr><tr><td>onPointerMove</td><td>The pointer is moving over a 3D element</td></tr><tr><td>onPointerOver (or onPointerEnter)</td><td>The mouse pointer is moved over a 3D element</td></tr><tr><td>onPointerOut (or onPointerLeave)</td><td>The mouse pointer moves out of a 3D element</td></tr><tr><td>onPointerDown</td><td>The mouse button is pressed over a 3D element</td></tr><tr><td>onPointerUp</td><td>The mouse button is released over a 3D element</td></tr></tbody></table>

```javascript
<mesh scale={10}
    onClick={(e, mesh)        => console.log( e.type, mesh ) }
    onContextMenu={(e, mesh)  => console.log( e.type, mesh ) }
    onDoubleClick={(e, mesh)  => console.log( e.type, mesh ) }
    onWheel={(e, mesh)        => console.log( e.type, mesh ) }
    onPointerMove={(e, mesh)  => console.log( e.type, mesh ) }
    onPointerEnter={(e, mesh) => console.log( e.type, mesh ) }
    onPointerLeave={(e, mesh) => console.log( e.type, mesh ) }
    onPointerDown={(e, mesh)  => console.log( e.type, mesh ) }
    onPointerUp={(e, mesh)    => console.log( e.type, mesh ) }
>
    <boxGeometry/>
    <meshStandardMaterial color={'red'}/>
</mesh>
```
