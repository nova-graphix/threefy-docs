# 5.2 Addons

The addons part of three.js supported by this threefy is as follows. Additions and updates will continue to be made in the future.

> **Note (v2.0 / WebGPU):** post-processing no longer runs on the WebGL-only `EffectComposer`. Threefy 2.0 drives effects through the node-based **`RenderPipeline`**, so the passes below are wired into a TSL pipeline internally — you still declare them as elements exactly as before. Three new passes are available under WebGPU: `<dotScreenPass/>`, `<rGBShiftPass/>`, and `<fXAAPass/>`.

| Category       | Threefy Elements                                                                                                                                                                                                                                               |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Controls       | <p>&#x3C;arcballControls /><br>&#x3C;dragControls /><br>&#x3C;firstPersonControls /><br>&#x3C;flyControls /><br>&#x3C;mapControls /><br>&#x3C;orbitControls /><br>&#x3C;pointerLockControls /><br>&#x3C;trackballControls /><br>&#x3C;transformControls /></p> |
