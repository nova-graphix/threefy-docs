# 6.4 Elements to support nature modeling

The following nature elements frequently appear in movies, games, etc., and are commonly used when composing scenes, so they are provided by threefy for user convenience. Performance will be improved through future upgrades, and more elements will be added.

> **Note:** These nature elements (`<Water>`, `<Sky>`, `<Terrain>`, `<Grass>`, and the others below) are **not yet exported by the `threefy` package** in the current v2 release — in the examples they run as demo-app helpers, and several also require additional image assets. Support is **planned for an upcoming release**.

```html
<Water
    waterSize={10000},
    textureWidth={512},
    textureHeight={512},
    waterNormals={'images/water/normal.jpg'},
    sunDirection={[0,0,0]},
    sunColor={0xffffff},
    waterColor={0x001e0f},
    distortionScale={3.7},
    size={1},
    fog={false},
/>
```

```html
<Sky
    skySize = 10000,
    turbidity = 10,
    rayleigh = 2,
    mieCoefficient = 0.005,
    mieDirectionalG = 0.8,
    sunPosition = [0,0,-1],
/>
```

```html
<SkyDome
    radius = 512,
    topColor = 0x0077ff,
    bottomColor = 0xffffff,
/>
```

```html
<Clouds
    amount = 1000,
    range = [ 1000, 150, 1000 ],
    size = [ 64, 64, 1 ],
    speed = 0.05,
/>
```

```html
<Terrain
    map = 'images/terrain/Rugged Terrain/diffuse.jpg',
    heightMap = 'images/terrain/Rugged Terrain/heightmap.jpg',
    xsize = 1024,
    ysize = 600,
    zsize = 1024,
/>
```

```html
<Grass
    heightMap      = undefined,
    numBlades      = 85000,
    bladeScale     = 1,
    color          = 0x737530,
    width          = 2048,
    hsize          = 0,
    transitionLow  = 0.31,
    transitionHigh = 0.36,
    windIntensity  = 1.5,
/>
```

```html
<SimpleTree
    leafColor = 0x3f6d21,
    leafScale = 1.0,
    trunkColor = 0x7c6344,
    windSpeed = 1.0,
/>
```

```html
<ProcTree
    leafDensity = 0.39,
    twigColor = 0x6db33f,
    twigHeight = 5,
    twigDirection = 0.0,
    twigStretch = 0.99,
    twigDensity = 5,
```
